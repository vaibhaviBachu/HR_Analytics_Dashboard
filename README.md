# HR_Analytics_Dashboard

Project Overview<br>
The HR Analytics Dashboard is an interactive Tableau visualization designed to help HR teams and management make data-driven decisions by providing insights into employee demographics, attrition, job satisfaction, and performance metrics. This project aims to highlight key patterns and trends to support workforce planning and improve employee retention strategies.<br>

<h1>Features</h1>

 Attrition Analysis: Understand which departments or job roles have the highest turnover rates.<br>

 Demographics Breakdown: Analyze employee distribution based on age, gender, education, and marital status.<br>

 Department-wise Insights: Drill down into HR metrics by department or job role.<br>

 Performance & Satisfaction: Correlate performance ratings with job satisfaction, overtime, and work-life balance.<br>

 Interactive Filters: Slice the data by department, education field, gender, etc.

<t1> Tools & Technologies</t1>
Tableau: For data visualization and interactive dashboard creation

Excel / CSV (assumed): For storing and preprocessing the HR data


 Key Insights
Departments with lower satisfaction levels also show high attrition rates.

Younger employees tend to switch jobs more frequently.

Overtime hours and low work-life balance scores strongly correlate with higher attrition.



 How to Use
Open the HR Analytics Dashboard.twbx file using Tableau Desktop.

Use the filters at the top to explore data by department, gender, and education.

Hover over charts to view detailed tooltips and insights.

Use dashboard actions for deeper drill-down analysis.
